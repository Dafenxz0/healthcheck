# Contributing

Thanks for considering a contribution.

## Local setup

```bash
python -m pip install -e .
python -m unittest discover -s tests
```

## Pull requests

- Keep changes focused.
- Include tests for new checks or output changes.
- Explain the maintainer problem the change solves.

## Issues

Bug reports are most useful when they include the repository shape being audited,
the expected result, and the actual result.
